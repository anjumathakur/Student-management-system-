# Student Management System

A web application to manage student records with real-time statistics and filtering.

## 🔗 Live Demo
https://anjumathakur.github.io/Student-Management-System/

## ✨ Features
- Add, search, and delete student records
- Live dashboard (total students, average marks, top score)
- Automatic grade calculation
- Filter by branch and academic year

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript (Vanilla)
